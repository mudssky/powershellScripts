请将刚才生成的计划保存为.claude/plan目录下的 PLAN.md 文件（如果是在为特定功能工作，请根据功能命名，如 PLAN_auth_feature.md）。

文件内容必须包含以下两部分：

## 第一部分：设计与思路 (Human Context)

* **目标概述**：我们要解决什么问题？
* **架构决策**：刚才讨论的实现方案细节、数据结构变更、关键逻辑说明。（保留 Plan Mode 中你分析的精华部分）
* **风险提示**：需要注意的坑或依赖关系。

## 第二部分：执行清单 (Machine Context)

* 将实施步骤转换为 Markdown Checkbox 列表。
* 格式要求：- [ ] 1. 具体的动作（涉及的文件路径）
* 粒度要细，适合单步执行。

注意：如果文件已存在，请询问是覆盖还是追加。
