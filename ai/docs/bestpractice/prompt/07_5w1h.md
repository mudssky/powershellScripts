# 5W1H：Who / What / When / Where / Why / How

5W1H 用于需求澄清与问题拆解，帮助模型先明确信息再给出答案。

## 适用场景

* 需求不清晰、输入信息不足。
* 希望模型先提问或梳理要点。
* 访谈、需求分析、问题复盘。

## 结构拆解

* Who：参与者/受众/责任人。
* What：要解决的问题/交付物。
* When：时间节点/时效要求。
* Where：应用场景/渠道/系统边界。
* Why：原因/目标/动机。
* How：方法/流程/资源。

## 模板

```text
Who: {角色/受众}
What: {要解决的问题}
When: {时间/期限}
Where: {场景/系统/渠道}
Why: {目标/动机}
How: {流程/方法/资源}
```

## 示例

```text
Who: 业务运营团队
What: 需要一份活动复盘报告
When: 本周五前
Where: 线上促销活动
Why: 评估转化效果并优化下一期
How: 汇总数据、对比目标、给出改进建议
```

## 注意事项

* 5W1H 可以先用于“提问清单”，再进入生成阶段。
* 当输入不完整时，明确哪些项缺失并提出补充问题。
