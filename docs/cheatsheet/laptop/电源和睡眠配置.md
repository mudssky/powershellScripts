这份 **笔记本电脑电源设置最佳实践 Cheatsheet (速查表)** 结合了效率、续航和硬件寿命保护。你可以直接参照此表对电脑进行微调。

---

### 核心原则 (The Golden Rules)

1. **缓冲原则：** 设置 **“睡眠时间” > “熄屏时间”**（至少间隔 2-5 分钟）。
    * *理由：避免阅读时屏幕一黑立刻断网/锁死，给用户反应时间。*
2. **防烧屏原则：** 无论是否插电，**必须设置“关闭显示器”** 时间。
    * *理由：保护屏幕（尤其是 OLED），延长背光寿命。*
3. **安全原则：** 必须开启 **“从睡眠/熄屏唤醒时需要密码”**。

---

### ⚡ 场景一：使用电池 (移动办公模式)

**目标：** 极度省电，防止意外耗尽，但不要频繁打断工作。

| 设置项 | 推荐值 | 说明 |
| :--- | :--- | :--- |
| **关闭显示器** | **2 - 3 分钟** | 屏幕是耗电大户，人走即灭。 |
| **进入睡眠** | **5 - 10 分钟** | 给自己留出“晃动鼠标救回屏幕”的缓冲期。 |
| **电源模式** | **最佳能效 / 省电模式** | 限制后台活动，降低CPU频率。 |
| **低电量设置** | **20% 自动开启节电模式** | Windows/Mac 均可设置，作为最后一道防线。 |
| **屏幕亮度** | **自动 / 50-60%** | 室内环境通常不需要 100% 亮度。 |

---

### 🔌 场景二：接通电源 (桌面主机模式)

**目标：** 性能全开，随时待命，保持后台任务运行。

| 设置项 | 推荐值 | 说明 |
| :--- | :--- | :--- |
| **关闭显示器** | **5 - 10 分钟** | 哪怕不睡眠，也要关屏幕保护硬件。 |
| **进入睡眠** | **2小时 或 永不** | **2小时**适合普通办公（午休自动睡）；<br>**永不**适合挂机下载、远程控制。 |
| **电源模式** | **最佳性能 / 平衡** | 释放硬件潜力。 |
| **硬盘休眠** | **20 分钟** | (高级设置) 机械硬盘建议设置，SSD影响不大。 |

---

### 💻 物理操作 (合盖与按钮)

**目标：** 建立明确的肌肉记忆。

| 动作 | 推荐设置 | 理由 |
| :--- | :--- | :--- |
| **关闭盖子** | **睡眠 (Sleep)** | **最重要设置**。塞进包里前必须确保它睡着，否则会过热甚至烧坏主板。 |
| **电源按钮** | **睡眠 或 关闭屏幕** | 按一下就休息，方便快捷。不建议设为关机（容易误触）。 |

---

### 🔋 电池健康保养 (长期寿命)

**目标：** 延缓锂电池物理老化。

* **黄金法则 (20-80法则)：** 尽量不要让电量长期低于 **20%**，也不要长期维持在 **100%**。
* **长期插电党 (关键)：**
  * 如果你 90% 的时间都插着电源，请在电脑品牌自带的控制软件中（如联想 Vantage, 华硕 MyASUS, Mac设置）开启 **“电池养护模式”** 或 **“限制充电至 80%”**。
  * *理由：长期满电会导致电池鼓包和容量衰减。*
* **温度控制：** 电池最怕热。打游戏或渲染时，垫高电脑底部散热。

---

### 🛡️ 特殊硬件特别注意

* **OLED 屏幕用户：**
  * **强制执行：** 接通电源时，熄屏时间不要超过 **5分钟**。
  * **建议：** 开启系统级的“隐藏任务栏”或使用黑色壁纸，防止烧屏。
* **带有机械硬盘的老本：**
  * 移动中严禁不关机/不睡眠。机械硬盘在读写时受到震动极易损坏数据。
