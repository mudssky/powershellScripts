# Prompt 编写框架与保存格式

本目录整理常见的 Prompt 编写框架，并补充可落地的保存格式说明，便于团队统一写法、复用与版本管理。

## 框架速览

- [01. RTF：Role / Task / Format](./01_rtf.md)
- [02. CRISP：Context / Role / Intent / Style / Parameters](./02_crisp.md)
- [03. CO-STAR：Context / Objective / Style / Tone / Audience / Response](./03_co-star.md)
- [04. RISEN：Role / Instructions / Steps / End goal / Narrowing](./04_risen.md)
- [05. CRAFT：Context / Role / Action / Format / Tone](./05_craft.md)
- [06. CARE：Context / Action / Result / Example](./06_care.md)
- [07. 5W1H：Who / What / When / Where / Why / How](./07_5w1h.md)

## 选择建议

* 任务明确、输出格式固定：优先 RTF / CRAFT。
* 需要更丰富上下文与风格约束：CRISP / CO-STAR。
* 多步骤任务或强约束交付：RISEN / CARE。
* 需求澄清或访谈型场景：5W1H。
