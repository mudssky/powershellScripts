# LLM 开发最佳实践指南

除了 Prompt 版本管理，LLM 应用开发与传统软件开发最大的区别在于：**核心逻辑从“确定性代码”变成了“概率性模型”**。

为了控制这种不确定性，以下是目前业界公认的 LLM 开发最佳实践，分为**架构、数据、评估、稳定性**四个维度。
