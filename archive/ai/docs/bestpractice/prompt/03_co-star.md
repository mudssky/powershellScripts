# CO-STAR：Context / Objective / Style / Tone / Audience / Response

CO-STAR 强调沟通与写作效果，尤其适合营销、对外沟通或教育类内容。

## 适用场景

* 内容写作、对外沟通、市场传播。
* 需要把“受众与语气”写清楚。
* 输出有明确结构或交付形式。

## 结构拆解

* Context：背景与素材。
* Objective：目的与目标效果。
* Style：表达风格与写作方式。
* Tone：语气（严谨、亲切、鼓励等）。
* Audience：目标受众与认知水平。
* Response：输出格式与长度。

## 模板

```text
Context: {背景/素材}
Objective: {想达成的目标}
Style: {写作风格}
Tone: {语气}
Audience: {受众}
Response: {输出格式/长度/结构}
```

## 示例

```text
Context: 我们上线了“团队协作看板”功能。
Objective: 让老用户理解新功能并愿意试用。
Style: 清晰、结构化。
Tone: 友好、鼓励。
Audience: 中小团队负责人。
Response: 200-250 字，分 2 段，结尾加一句行动号召。
```

## 注意事项

* Objective 要写“期望行为”，比如“注册/试用/完成下载”。
* Audience 明确岗位或人群画像，避免“所有用户”。
