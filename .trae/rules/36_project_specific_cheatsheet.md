# 📂 Project Specific Rules (Cheatsheets)

## 1. Format

- **Markdown**: 标准 Markdown 格式。
- **Frontmatter**: 可选，用于元数据。

## 2. Content Guidelines

- **Concise**: 拒绝废话，直接给出命令或代码片段。
- **Copy-Paste Ready**: 代码块必须可直接运行。
- **Structure**:
  - `## Installation`
  - `## Common Commands`
  - `## Configuration`
  - `## Troubleshooting`

## 3. Organization

- 按技术栈/工具分类目录 (e.g., `git/`, `linux/`, `pwsh/`)。
- 文件名使用 `kebab-case.md` 或 `Subject.md`。
